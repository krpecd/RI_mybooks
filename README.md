# MyBooks
React application that lets you categorize your books in shelfs.
App pages:
 - mainPage - Here you can see your shelfs and manage the books in them.
 - searchPage - Page for searching books. You can add books from there to your shelf. 

## Tech stack
- React
- Typescript
- data fetching via Fetch API
- plain CSS for styling

## Getting started
First install dependencies
```bash
npm
```
Than run the development server (running on port 3000):
```bash
npm start
```

## Building for production
For production we have to generate optimized build.
```
npm run build
```
